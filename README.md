# 📝 Online-Leave-Notification
การแจ้งลางานผ่านเว็บไซต์ออนไลน์

ตอนออกฝึกสหกิจศึกษา ได้ทำ Project การแจ้งลางานผ่านเว็บไซต์ออนไลน์  
โดยพัฒนาในรูปแบบเว็บแอปพลิเคชันด้วยภาษา PHP เชื่อมต่อฐานข้อมูล MySQL และจัดการฐานข้อมูลผ่าน phpMyAdmin ผู้ใช้งานสามารถเข้าสู่ระบบ แจ้งลางาน ตรวจสอบสถานะการลา และให้ฝ่าย HR หรือหัวหน้า HR ตรวจสอบพร้อมอัปเดตสถานะการลาได้

## 🛠️ เทคโนโลยีที่ใช้

- `PHP` สำหรับพัฒนา
- `MySQL / MariaDB` สำหรับจัดเก็บข้อมูลผู้ใช้งานและข้อมูลการลา
- `phpMyAdmin` สำหรับสร้างและจัดการฐานข้อมูลจำลอง
- `HTML` สำหรับโครงสร้างหน้าเว็บ
- `CSS` สำหรับตกแต่งหน้าตาเว็บไซต์
- `JavaScript` สำหรับการสลับแท็บและโต้ตอบหน้าเว็บเบื้องต้น
- `jQuery UI Datepicker` สำหรับช่วยเลือกวันที่ในบางหน้ารายงาน
- `LINE Notify API` สำหรับส่งการแจ้งเตือนเมื่อมีการแจ้งลา
- `Google Fonts` และ `Font Awesome` สำหรับฟอนต์และไอคอนบนหน้าเว็บ

## 🏗️ ความสามารถหลักของระบบ

- ล็อกอินเข้าสู่ระบบด้วยรหัสพนักงานและวันเดือนปีเกิด
- แยกสิทธิ์การใช้งานตามบทบาทผู้ใช้
- กรอกแบบฟอร์มแจ้งลางานผ่านเว็บไซต์
- เลือกประเภทการลา เช่น ลากิจ ลาป่วย และอื่น ๆ
- บันทึกข้อมูลการลาลงฐานข้อมูล
- ส่งแจ้งเตือนไปยัง LINE เมื่อมีการแจ้งลา
- ตรวจสอบประวัติและสถานะการลาย้อนหลังได้
- HR สามารถอนุมัติ ยกเลิก หรือปรับสถานะการลาได้
- Head of HR สามารถกำหนดสิทธิ์ผู้ใช้งานเพิ่มเติมได้
- มีหน้ารายงานและสถิติการลาของพนักงาน

## 👥 ระดับผู้ใช้งานในระบบ

- `User` สำหรับพนักงานทั่วไป ใช้แจ้งลาและตรวจสอบสถานะของตนเอง
- `HR` สำหรับเจ้าหน้าที่ HR ใช้ตรวจสอบและอัปเดตสถานะการลา
- `Head of HR` สำหรับหัวหน้า HR ใช้ตรวจสอบข้อมูลทั้งหมดและจัดการสิทธิ์ผู้ใช้งาน
